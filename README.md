# ETHTokyo 2026 official website

## Features

- Typescript
- [HonoX](https://github.com/honojs/honox)
- [Vite](https://vite.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Biome](https://github.com/biomejs/biome) linter & formatter

## Develop

```bash
# develop
pnpm install
pnpm dev

# build & preview
pnpm run preview
```
